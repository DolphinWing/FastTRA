操作方式：

1. 先在選項中輸入想要訂的資料
2. 打開台鐵的訂票網頁，套件會在網頁中新增兩個按鈕
3. 在要訂票的時候按"快速去程"或"快速回程"，套件會自動幫你選取在選項中設定的資料
4. 繼續使用台鐵訂票網站(輸入驗證碼等後續操作)
